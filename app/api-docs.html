REST API Reference

Full programmatic access to all 28 data generators. No auth required. Works from any client.

Base URL

local development
http://localhost:8080
Note: The API is served by the same server.py that serves the app. Start it with python server.py (or start.bat on Windows).

Endpoints

GET /api/categories
List all categories and the types they contain.
example response
{
  "categories": [
    { "id": "identifiers", "name": "Identifiers & Security", "icon": "🔑",
      "types": ["uuid", "hash", "password", "username"] },
    { "id": "network",     "name": "Network & Web",          "icon": "🌐",
      "types": ["ip", "url", "mac_address", "imei"] },
    // ... 5 more categories
  ]
}
GET /api/types
Flat list of all supported type IDs.
example response
{ "types": ["uuid", "hash", "password", "username", "name", "email", ...28 total] }
GET /api/generate?type={type}&count={n}&[options]
Generate data via query string. All type-specific options can be appended as extra parameters.
curl examples
curl "http://localhost:8080/api/generate?type=uuid&count=5"
curl "http://localhost:8080/api/generate?type=password&count=3&length=24&special=true"
curl "http://localhost:8080/api/generate?type=credit_card&count=2&card_type=Visa"
curl "http://localhost:8080/api/generate?type=mac_address&count=5&separator=-&uppercase=false"
curl "http://localhost:8080/api/generate?type=ip&count=4&version=ipv6"
curl "http://localhost:8080/api/generate?type=date&count=5&format=us&from_year=1990&to_year=2024"
POST /api/generate
Generate data via JSON body. Preferred for complex options.
request body schema
typestring*The data type ID (see full list below). Required.
countintegerNumber of values to generate. Default: 10. Max: 1000.
optionsobjectType-specific options (see per-type table below). Optional.
curl examples
curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"type":"password","count":5,"options":{"length":32,"special":true}}'

curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"type":"address","count":3,"options":{"country":"JP"}}'

curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"type":"credit_card","count":10,"options":{"card_type":"Mastercard","valid":"valid"}}'
response schema
typestringThe requested type ID.
countintegerNumber of items returned.
optionsobjectOptions that were applied.
datastring[]Array of generated values.

All 28 Supported Types

🔑 Identifiers & Security

uuid options
prefix: string  — embedded at the start of the UUID hex
suffix: string  — embedded at the end of the UUID hex
hash options
algorithm: "md5" | "sha1" | "sha256"   (default: "sha256")
password options
uppercase: true|false   (default: true)
lowercase: true|false   (default: true)
numbers:   true|false   (default: true)
special:   true|false   (default: false)
length:    integer      (default: 16, range: 4–128)
username options
style:  "name_year" | "adj_noun" | "name_random" | "mrx"   (default: "name_year")
prefix: string

👤 Contact & Identity

name options
starts_with: string
ends_with:   string
email options
domain:    string
extension: "com" | "org" | "net" | "io" | "test" | "co"
phone options
country:      country code e.g. "US", "IN", "GB"   (default: "US")
include_code: true|false   (default: true)
address / city / zipcode options
country: country code e.g. "US", "DE", "JP"
coordinates options
precision: integer   decimal places (default: 6, range: 1–8)

💳 Financial & Sensitive

credit_card options
card_type: "Visa" | "Mastercard" | "American Express" | "Random"
valid:     "valid" | "invalid"
ssn options
country: "US" | "UK" | "Random"   (default: "US")
barcode options
numeric_only: true|false   (default: true)
length:       integer      (default: 13, range: 8–20)
isbn options
format: "isbn10" | "isbn13"   (default: "isbn13")

🌐 Network & Web

ip options
version: "ipv4" | "ipv6"   (default: "ipv4")
url options
protocol:  "https" | "http"
domain:    string
extension: "com" | "net" | "org" | "io" | "test" | "co"
mac_address options
uppercase: true|false   (default: true)
separator: ":" | "-"    (default: ":")
imei options
brand:          "Apple" | "Samsung" | "Xiaomi" | "Generic"
valid_checksum: true|false   (default: true)

🕐 Time & Text

date options
from_year: integer   (default: 2000)
to_year:   integer   (default: 2030)
format:    "iso" | "us" | "eu"   (default: "iso")
datetime options
include_date:     true|false   (default: true)
include_time:     true|false   (default: true)
include_timezone: true|false   (default: false)
sentence options
grammatically_valid: true|false   (default: true)
paragraph options
min_sentences: integer   (default: 3)
max_sentences: integer   (default: 6)

🎨 Colors

hex_color options
uppercase: true|false   (default: true)
rgb_color options
min_value: integer   (default: 0,   range: 0–255)
max_value: integer   (default: 255, range: 0–255)

🏢 Work & Organization

company options
starts_with: string
job options
seniority: "any" | "junior" | "senior" | "lead"   (default: "any")
